Tools & Git

Comandos Git

  • git clone: clona um repositório existente(remoto) para o computador(local).
  • git status: mostra o estado do repositótio e quais ficheiros foram modificados, adicionados ou removidos.
  • git add: adiciona uma alteração no diretório activo à área de staging.
  • git restore: reverte ou descarta mudanças feitas em ficheiros que tenham sido adicionado à área de staging.
  • git commit -m "mensagem": guarda as alterações feitas e adiciona uma mensagem.
  • git log: exibe o histórico de commits do repositório.
  • git push: faz envio das alterações do repositório local para o remoto.
  • git pull: descarrega conteúdos de um repositório remoto e atualiza no local.
  • git branch: cria uma cópia independente do código sem interferir no repositório principal.
  • git checkout: serve para criar e/ou trocar de branches/commits.
  • git merge: combina duas ou mais branches do projecto em uma única branch.
  • git fork: cria uma cópia completa de um código já existente, de forma a trabalhar nele de forma independente.
